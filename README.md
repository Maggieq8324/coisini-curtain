美佳布艺 - 窗帘电商小程序

项目架构：Java API + CMS + 小程序

#### 项目地址
CMS：[美佳窗帘布艺 - https://www.meijia.pub](https://www.meijia.pub)
<br>
小程序：美佳布艺（微信搜一搜）

<br>
